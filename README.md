\# 🍕 Автотесты для сайта пиццерии



UI автотесты для интернет-магазина пиццерии на Python + Selenium.



\## 🛠 Стек технологий

\- Python 3

\- Selenium WebDriver

\- pytest

\- Allure Reports

\- Page Object Model



\## 📁 Структура проекта

\- `tests/` — тестовые сценарии

\- `pages/` — Page Object классы

\- `utils/` — вспомогательные утилиты

\- `logs/` — логи выполнения



\## 🚀 Как запустить

1\. Установить зависимости:

&#x20;  pip install -r requirements.txt

2\. Запустить тесты:

&#x20;  pytest tests/

3\. Открыть Allure-отчёт:

&#x20;  allure serve allure-results



\## ✅ Что тестируется

\- Главная страница

\- Меню и выбор пиццы

\- Корзина

\- Применение купонов и бонусов

\- Оформление заказа
